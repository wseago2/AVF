Visit Programmableweb.com and search for JSON-P data sources. Why would we need JSON-P, specifically?

JSON data is a very universal, easy to use data exchange format, but it violates the Same Origin Policy. JSONP uses a callback function to get around this. This is why we need JSONP.

Please describe any design and functionality intentions you may have for your Demo app components this month.

Unfortunately, my time is very limited, so I have to work with functionality taking priority over design and polish. In the real world, I believe that design outweighs functionality, especially in a world where apps tend to get more and more complex as time goes by. I think it is very important to remember that well designed simplicity will always be more desirable than all-encompassing mediocrity.


I am looking forward to getting hands-on with Phonegap. I believe it represents a valid way for people at my stage of the MDVBS program to actually begin to create viable applications. My ultimate goal for this project is to create an astronomy app that will help me in my hobby, as well as lay a foundation for many astronomy related apps and technologies in the future.


Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

Probably the biggest lesson I have learned is to try to spend as much time as possible in the planning stage of the application development cycle. Things tend to go much smoother that way.
